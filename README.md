# NetUsage Monitor

یک ابزار سبک، بهینه و متن‌باز برای مانیتورینگ لحظه‌ای مصرف اینترنت (آپ‌لود و دانلود) با قابلیت ذخیره‌سازی در دیتابیس SQLite.

## ویژگی‌ها
- **مانیتورینگ لحظه‌ای:** نمایش سرعت آپلود و دانلود با استفاده از کتابخانه `psutil`.
- **ذخیره‌سازی هوشمند:** ثبت لاگ‌ها در دیتابیس SQLite با قابلیت تجمیع داده‌های روزانه.
- **رابط کاربری زیبا:** استفاده از `rich` برای نمایش داده‌ها در ترمینال.
- **خروجی‌گیری:** امکان اکسپورت داده‌ها به فرمت CSV.

## پیش‌نیازها
- Python 3.9+
- Poetry (برای مدیریت وابستگی‌ها)
